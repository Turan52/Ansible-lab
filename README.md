A collection of Ansible playbooks built and tested on Ubuntu and Fedora (Red Hat based) systems. Written while learning how Ansible works, covering tasks like package management, service configuration and system automation. The playbooks are organized by topic and work across both Debian based and RPM based distributions. Built and run from the command line using standard Ansible CLI tools. Feel free to clone, explore and adapt any playbook to your own setup.
